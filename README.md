# List of simple and useful MacOS apps and extensions

- [Mater](https://github.com/jasonlong/mater) - A minimal Pomodoro timer that lives in your macOS menu bar.
- [MusicDecoy](https://github.com/FuzzyIdeas/MusicDecoy) - Stop launching the Music app whenever you press ▶ Play (it's especially relatable for non-apple earphones)
- [BlackHole](https://github.com/ExistentialAudio/BlackHole) - virtual audio loopback, useful for [SoundPad](https://github.com/aks1n3d/SoundPad) and for routing audio from DAWs.
- [MacPacker](https://github.com/sarensw/MacPacker) - preview archives without extracting them.
- [MiddleClick](https://github.com/artginzburg/MiddleClick) - "wheel click" or three-finger click/tap
- [TouchHLE](https://github.com/touchHLE/touchHLE) - High-level emulator for old iPhone OS apps.
- [qemu-t8030](https://github.com/TrungNguyen1909/qemu-t8030) - iPhone 11 emulated on QEMU
- [super-tart-vphone-writeup](https://github.com/wh1te4ever/super-tart-vphone-writeup) - virtual iPhone that runs natively on M-seris Macs
